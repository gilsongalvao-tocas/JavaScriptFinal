 Public API Dashboard

Final JavaScript Project built by Gilson Rabelo Galvao.

## Project Overview

This project is a single-page dashboard made with HTML, CSS, and vanilla JavaScript. It includes multiple mini apps that fetch live data from public APIs and display the results on the page.

## Features

- Random dog image
- Random cat image
- Weather search by city
- USD to EUR currency converter
- GitHub user search
- Random joke
- Random advice
- TV show search
- Responsive dashboard layout
- Black, white, and yellow color theme

## APIs Used

- Dog CEO API
- The Cat API
- Open-Meteo API
- ExchangeRate API
- GitHub API
- JokeAPI
- Advice Slip API
- TVMaze API
